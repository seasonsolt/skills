# 人设定位（简介）

简介是把"主页访客"转成"关注"的开关。很多账号访客不少却涨粉慢，就是因为简介只说了"我是谁"，没给"关注我有什么用"。

## 三行公式

```
第1行 人设   —— 你是谁，一个有记忆点/有态度的身份钩子
第2行 定位   —— 你写什么，点名用户会去搜的关键词（利于被搜索命中）
第3行 关注理由 —— 更新频率承诺 + 行动引导（关注不迷路 / 系列持续更）
```

**示例**（通用范式，换成你自己的赛道）：
```
🍳 十年后厨出身的家常菜作者          ← 人设：一句有记忆点的身份
📌 只写在家复刻得出的餐厅菜：粤菜 / 卤味 / 家宴  ← 定位：点名可被搜索的关键词
🔔 每周一道菜谱，关注不迷路           ← 关注理由：更新承诺 + 行动引导
```

## 要点与坑

- ⚠️ **简介只能在手机 App 改**（我 → 编辑资料 → 简介），网页版和创作平台都没有编辑入口。所以你的产出是**一段可直接粘贴的文案**，让用户去 App 粘。
- **别自夸"高质量/优质"**——质量由读者判断，自夸显心虚。承诺**具体内容 + 频率**更有说服力。
- **承诺要能兑现**：写了"每周一篇"就得做到，否则伤人设。做不到就写"持续更新"。
- **emoji 每行行首一个**当图标提高扫读性，行内别乱塞。纯文字版也成立（更像技术人），属风格偏好。
- 简介的关键词要和账号**已验证的爆款方向**一致，别写成什么都做。

## 配套动作

改完简介，顺手把**最好的一篇笔记设为置顶**（网页版笔记管理悬停笔记有置顶按钮），让新访客第一眼看到你的代表作——这一步和简介一起补齐"关注转化"。
